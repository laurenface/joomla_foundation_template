Hey there,

The description says it all. This is a Joomla template that has responsiveness built into it using the Foundation CSS framework. When you unzip a Joomla 1.7 download, there is a default template "atomic". We have built on top of this template as a way to show that responsiveness can also be used for your websites.

Screenshot

How to Install

  • -Download the repo.
  • -Unzip the repo into /your_joomla_site/templates/
  • -Make sure the template files are at /your_joomla_site/templates/new_atomic/
  • -Login to the Administration of your joomla site
  • -Click "Extensions > Extension Manager"
  • -Click the "Discover" link, then click "Purge Cache".
  • -Click the "Discover" button.
  • -You will see the newly added template "new_atomic" in the list.
  • -Finally, click "Install".

There, all done!
